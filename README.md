# pdPatches

## Description
This is a collection of useful pure data patches.
It includes a set of abstractions, demos and some standalone patches.

This is still a work in progress. 

## Abstractions
* tgDSP: toggle for pd's DSP
* tgMaster~: stereo gain with vu meters and mute toggle, connected to the dac
* tgRecorder~: stereo recorder
* tgMetro: wrapper of the metro object with on/off toggle, visualises impulses
* tgStepsequencer2, tgStepsequencer4, tgStepsequencer8: stepsequencers with toggles to randomise or reverse the order
* tgOSCSend: wraps of updsend with on/off toggle and port number box, currently only sends to localhost
